## ⚡ 快速命令

```bash
# 激活虚拟环境
source .venv/bin/activate         # macOS/Linux
.venv\Scripts\activate            # Windows

# 启动应用
python main.py
# 或
./start.sh                        # macOS/Linux
start.bat                         # Windows

# 运行测试
python test.py

# 本地构建
./build.sh                        # macOS/Linux
build.bat                         # Windows

# 安装依赖
pip install -r requirements.txt
```
